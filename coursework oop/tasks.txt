Опис результату роботи програми:

1. Відкривається консоль.
2. Пропонується обрати одну із запропонованих книг (введенням числа) або додати свою (користувач має вписати шлях до книги).
3. Пропонується ввести перше слово.
4. Пропонується ввести кількість слів в реченні.
5. Користувач отримує результат.

Що для цього треба: 

+ Додати змінну, яка приймає кількість слів в реченні з консолі
+ Додати можливість вибору книги:
	1. Додати змінну, яка приймає введене користувачем число. 
	2. Знайти потрібну книгу та передати значення в потрібний клас для обробки.
+ Додати можливість додавання книги
+ Додати перевірку на правильність вводу даних користувачем
	1. + При введенні не існуючої книги - вивести текст та повторити спробу ще 2 рази. Потім вивести 1 продовжити 0 вийти.
	2. + При введені не правильного шляху - повторити спробу ще 2 рази. Потім вивести 1 продовжити 0 вийти.
	3. + При введенні ключового слова - перевірка на слово. Якщо false - вивести повідомлення та повторити спробу ще 2 рази. Потім вивести 1 продовжити 0 вийти.
	4. + При введенні кількості слів - перевірка на ціле число. Якщо false - вивести повідомлення та повторити спробу ще 2 рази. Потім вивести 1 продовжити 0 вийти.
